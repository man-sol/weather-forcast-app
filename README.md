# Weather Forecast Application

A responsive and interactive Weather Forecast Application built using JavaScript, HTML, CSS, and Tailwind CSS. This app fetches real-time weather data using the OpenWeatherMap API and displays current weather and 5-day forecasts with a sleek black and yellow theme.


## Features

- Search by City Name: Users can search for weather in any city.  
- Current Location Weather: Fetch weather for the user's current location using geolocation.  
- 5-Day Forecast: View an extended 5-day forecast with temperature, humidity, wind, and weather icons.  
- Temperature Toggle: Switch between Celsius (°C) and Fahrenheit (°F) for today's temperature.  
- Recent Searches: Dropdown menu with recently searched cities stored in localStorage.  
- Weather Alerts: Custom alerts for extreme temperatures (above 40°C).  
- Dynamic Background: Background changes based on current weather (clear, clouds, rain).  
- Responsive UI: Fully responsive for desktop, tablet, and mobile devices.  
- Error Handling: Validates user input and gracefully handles API errors with proper messages.  
- Local Weather Icons: Uses custom icons for consistent display across all forecasts.
